# LumenWrite UI Template

Este é o seu template de front-end extraído do LumenWrite. Ele contém toda a identidade visual premium, componentes de UI e estrutura de layout prontos para uso.

## Como começar um novo projeto

1.  **Copie esta pasta** para o local do seu novo projeto.
2.  Abra o terminal na pasta e instale as dependências:
    ```bash
    npm install
    ```
3.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```

## Estrutura do Template

- `src/index.css`: Contém o Design System (fontes, cores, sombras).
- `src/components/ui/`: Componentes básicos reutilizáveis (Button, Card, Modal).
- `src/components/Layout.tsx`: Estrutura principal (Sidebar + Header).
- `src/App.tsx`: Exemplo de como usar os componentes.

## Tecnologias inclusas
- **React 19**
- **Vite**
- **Tailwind CSS 4**
- **Lucide React** (Ícones)
- **Motion** (Animações)
- **TypeScript**
